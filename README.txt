# Scrapy-ProPub
The modifed Scrapy files utilized to obtain data from the, "Dollars for Docs" article featured on propublica.org. 

This yields the inital 40 pages (25 rows per page) of results, and returns the following data:
  Payee - Doctor or institiuion that accepted Pharmecutical money. 
  Url - Link to specific page.
  City - Payee's city.
  State - Payee's state.
	Company - Pharmecutial company that made the payment. 
	Amount -  Total amount (usd) given.
